1. KN University Network Design

This project involved designing a secure, multi-campus network infrastructure for KN University. The network design included VLANs, firewalls, and IoT integration for enhanced scalability and security. Active Directory was deployed with Role-Based Access Control (RBAC) to ensure user-specific permissions and secure access control.

Additionally, cloud solutions were implemented to support scalability and prototype physical network configurations. This project demonstrated my ability to design complex network architectures and manage secure access control protocols.

2. Smart Home System (CQ Smart Systems)

The CQ Smart Systems project was a cloud-based IoT home management system developed using Azure IoT Hub and Node-RED. The system allowed for real-time monitoring and management of various home devices, providing users with a centralized control interface.

This project utilized Kubernetes and Docker for containerized deployment and scalability. Security protocols were implemented to safeguard the data and operations of connected devices, ensuring robust privacy and data protection measures. This project highlighted my skills in IoT, cloud computing, and containerization.

4. Facial Recognition Attendance System

The Facial Recognition Attendance System was built as a real-time solution for automated attendance tracking. Using OpenCV and the Histogram of Oriented Gradients (HOG) technique for face recognition, the system ensured accurate identification of individuals.

Privacy compliance was a priority in this project, and various measures were taken to ensure the security of captured data. This system demonstrated my expertise in computer vision, face recognition technology, and the development of privacy-compliant solutions.

5. Cryptography Security Project

This project focused on secure communication by implementing Diffie-Hellman key exchange and AES encryption. The project aimed to establish a secure channel between client and server applications, ensuring data confidentiality and integrity.

The implementation involved creating encryption protocols to secure data transmission and included analysis of the encryption and key exchange processes. This project demonstrated my understanding of cryptographic algorithms, secure communication protocols, and their practical applications in cybersecurity.
